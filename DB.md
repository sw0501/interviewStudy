## 질문 대답

### 1. 데이터베이스의 특징

> 데이터베이스는 4가지 특징을 가진다.

1. 실시간 접근성 : 비정형적인 질의에 대하여 실시간 처리에 의한 응답이 가능하다.
2. 지속적인 변화 : 새로운 데이터의 삽입, 삭제, 갱신을 통해 항상 최신의 데이터를 유지한다.
3. 동시 공용 : 다수의 사용자가 동일한 데이터를 이용할 수 있어야 한다.
4. 내용에 의한 참조 : 사용자가 질의한 쿼리문에 해당하는 내용을 바탕으로 데이터를 조회한다.
